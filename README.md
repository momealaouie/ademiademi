# AB Ademi & Ademi

Ny responsiv företagswebbplats för AB Ademi & Ademi i Helsingborg. Sajten presenterar företagets tjänster inom paketering, montering och etikettering.

## Lokal utveckling

```bash
npm install
npm run dev
```

Öppna [http://localhost:3000](http://localhost:3000).

## Kvalitetskontroll

```bash
npm run lint
npm run typecheck
npm run build
```

## Publicering

Projektet är byggt med Next.js och är förberett för automatisk publicering via Vercel när GitHub-repot kopplas till ett Vercel-projekt.
